g2k_lib.metrics module¶

g2k_lib.metrics.get_error(num, denom)¶

Returns a percentage ratio between the two given array_like parameters according to their standard deviation.

Parameters:
  • num (array_like) – The numerator.
  • denom (array_like) – The denominator.
Returns:

The percentage ratio.
Return type:

float
g2k_lib.metrics.norm(x, L=2)¶

Returns the L-norm of a matrix.

Parameters:
  • x (array_like) – The matrix whose norm wants to be computed.
  • L (float or numpy.inf) –
    The order of the norm:
    • L=0 : returns the amount of non zero values in x.
    • L=numpy.inf : returns the maximum value in x.
    • else : standard L-th norm.
Returns:

The computed norm.
Return type:

float
